Función: Palíndromo

Descripción: Verifica si una palabra se lee igual de derecha a izquierda como de izquierda a derecha.
Parámetros: Variable booleana por referencia en la que se guardará si es verdadero o falso el caso que sea palindromo.
Valor de retorno: El valor de verdad del caso.


Función: Múltiplo

Descripción: Verifica si un número es par o impar sin usar mod %.
Valor de retorno: El valor de verdad del caso.


Función: Factorial

Descripción: Calcula el factorial de un número.


Función: Main

Descripción: Pregunta al usuario la opción que desee para así realizarla, se termina cuando el usuario teclea 0.
Valor de retorno: 0 al terminar el programa.
